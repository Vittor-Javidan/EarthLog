# These inputs are design to be used only by Widgets and PseudoWidgets.

## Checklist when creating a new input:
  - Add input data on ProjectService
  - Check if all input rules are working
  - Add the input on `IDService.changeIDsByReference_Input`
  - Add on `DOCX` export document
  - Add on `CSV` export document if applicable
  - Add all translations on `translations.ts`
